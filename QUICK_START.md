## Установка и запуск

### Предварительные требования

- **Node.js** 18+
- **npm** 9+ или **yarn** 1.22+
- Современный браузер (Chrome, Firefox, Safari, Edge)

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/fastfingers.git

# Перейти в директорию
cd fastfingers

# Установить зависимости
npm install
```

### Запуск в режиме разработки

```bash
# Запустить dev сервер
npm run dev

# Приложение будет доступно по адресу:
# http://localhost:5173
```

### Сборка для production

```bash
# Создать production build
npm run build

# Предпросмотр production build
npm run preview
```

### Запуск тестов

```bash
# Запустить все тесты
npm test

# Запустить тесты в watch режиме
npm run test:watch

# Запустить тесты с покрытием
npm run test:coverage
```

### Линтинг и форматирование

```bash
# Проверить код с ESLint
npm run lint

# Исправить проблемы автоматически
npm run lint:fix

# Форматировать код с Prettier
npm run format
```

## 📖 Основные функции

### 1. Режим обучения 📚

Пошаговое обучение слепой печати:

1. Откройте приложение
2. Нажмите "Режим обучения" или `Ctrl+L`
3. Выберите урок (начните с "Основной ряд")
4. Следуйте инструкциям и практикуйтесь

**7 уроков:**

- Основной ряд (ФЫВА ОЛДЖ)
- Верхний ряд (ЙЦУК ЕНГШ)
- Нижний ряд (ЯЧСМ ИТЬБ)
- Цифры (1234567890)
- Знаки препинания
- Слова
- Предложения

### 2. Кастомные упражнения ✏️

Создавайте собственные тексты:

1. Нажмите "Кастомные упражнения"
2. Кликните "Создать"
3. Введите название и текст
4. Выберите сложность (1-5)
5. Добавьте теги (опционально)
6. Нажмите "Создать"

### 3. Горячие клавиши ⌨️

**Навигация:**

- `Escape` - Закрыть окно
- `Ctrl+N` - Новое упражнение
- `Ctrl+R` - Перезапустить
- `Ctrl+S` - Статистика
- `Ctrl+L` - Режим обучения

**Печать:**

- `Ctrl+Space` - Пауза/Продолжить
- `Ctrl+Enter` - Завершить

**Настройки:**

- `Ctrl+,` - Открыть настройки
- `Ctrl+T` - Сменить тему
- `Ctrl+M` - Вкл/Выкл звук

**Общее:**

- `Shift+?` - Показать горячие клавиши
- `Ctrl+K` - Командная палитра

### 4. Ежедневные челленджи 🎯

Выполняйте задания каждый день:

1. Откройте "Челлендж дня"
2. Посмотрите цель (WPM, точность, слова и т.д.)
3. Практикуйтесь для достижения цели
4. Получите награду при выполнении

**Типы челленджей:**

- Спринтер (достичь WPM)
- Снайпер (достичь точности)
- Марафонец (напечатать слов)
- Выносливость (тренироваться время)
- Комбо мастер (достичь комбо)

### 5. Достижения 🏆

Разблокируйте 19 достижений:

**Категории:**

- Скорость (10, 30, 50, 70, 100 WPM)
- Точность (90%, 95%, 98%, 100%)
- Практика (слова, сессии)
- Серии (3, 7, 30, 100 дней)
- Особые

**Уровни редкости:**

- Обычное
- Редкое
- Эпическое
- Легендарное

### 6. Живая статистика 📊

Отслеживайте метрики в реальном времени:

- **WPM** - скорость печати
- **Точность** - процент правильных нажатий
- **Ошибки** - количество ошибок
- **Время** - время сессии
- **Слова** - напечатанные слова
- **Комбо** - правильные нажатия подряд

## 🎨 Персонализация

### Темы оформления

1. Откройте настройки (`Ctrl+,`)
2. Выберите "Тема"
3. Выберите светлую или тёмную тему

### Звуковые темы

1. Откройте настройки
2. Выберите "Звук"
3. Выберите тему:
   - Механическая клавиатура
   - Печатная машинка
   - Мягкая клавиатура
   - Футуристическая
   - Без звука

### Раскладки клавиатуры

Поддерживаемые раскладки:

- ЙЦУКЕН (Russian)
- QWERTY (English)
- DVORAK (English)

## 💾 Экспорт и импорт данных

### Экспорт

1. Откройте "Экспорт/Импорт"
2. Нажмите "Экспортировать данные"
3. Файл будет сохранён в формате JSON

**Что экспортируется:**

- Статистика
- Достижения
- Настройки
- Прогресс
- Цели
- Кастомные упражнения

### Импорт

1. Откройте "Экспорт/Импорт"
2. Нажмите "Импортировать данные"
3. Выберите файл JSON
4. Подтвердите импорт

⚠️ **Внимание:** Импорт перезапишет все текущие данные!

## 🎮 Режимы практики

### Режим практики

Стандартный режим для тренировки:

- Без ограничения времени
- Фокус на точности
- Подходит для обучения

### Режим спринта

60 секунд на максимум:

- Ограничение по времени
- Фокус на скорости
- Соревновательный элемент

### Режим марафона

Длительная практика:

- Большой объём текста
- Фокус на выносливости
- Отслеживание прогресса

**Q: Как сбросить прогресс?**
