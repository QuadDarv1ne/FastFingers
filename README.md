## 📖 О проекте

**FastFingers** — это веб-платформа для обучения слепой печати, которая сочетает интерактивный интерфейс, игровую механику и возможности искусственного интеллекта для создания персонализированной траектории обучения.

Мы делаем процесс обучения не просто полезным, но и увлекательным, помогая пользователям достичь высокой скорости печати и снизить утомляемость при работе за компьютером.

## ✨ Ключевые возможности

### Интерактивный тренажёр

- Визуализация виртуальной клавиатуры с подсветкой текущей клавиши и зон ответственности пальцев
- **Мгновенная обратная связь:** зелёный — правильно, красный — ошибка
- **Звуковое сопровождение** — приятные звуковые сигналы при правильном/неправильном вводе
- **Поддержка раскладок:** QWERTY, ЙЦУКЕН, Dvorak
- Автоматическая генерация текстов разной сложности

### ⚡ Режим спринта

- **60 секунд на максимум** — проверь свою скорость печати на время
- **WPM в реальном времени** — отслеживай свой прогресс во время спринта
- **Соревновательный элемент** — бей свои рекорды

### 📱 Адаптивность и PWA

- Полная поддержка всех устройств: от ПК до смартфонов
- **Progressive Web App** — установка как нативного приложения
- Офлайн-режим для тренировок без интернета

### 🎮 Геймификация

- Система уровней, достижений и наград
- Ежедневные задания и челленджи
- XP за каждую тренировку
- Прогресс-бар уровня

### 📊 Детальная статистика

- Графики скорости (WPM, CPM) и точности
- **Тепловая карта ошибок** — визуализация проблемных клавиш
- **История тренировок** — отслеживай свой прогресс по дням
- Статистика за 24 часа, 7 дней, 30 дней
- Лучшие результаты

### 👥 Категории упражнений

- **Основной ряд** — отработка базовых клавиш
- **Верхний ряд** — тренировка верхнего ряда клавиатуры
- **Нижний ряд** — тренировка нижнего ряда клавиатуры
- **Слова** — практика на реальных словах
- **Предложения** — сложные предложения для продвинутых
- **Код** — примеры кода для программистов

## 🚀 Быстрый старт

### Требования

- Node.js 18+
- npm 9+

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/fastfingers.git
cd fastfingers

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
# Создание production-сборки
npm run build

# Предпросмотр сборки
npm run preview
```

## 📱 Кроссплатформенная сборка

FastFingers поддерживает сборку для всех популярных платформ:

### 🌐 Web / PWA
```bash
npm run build
```
PWA готово к развертыванию на любом хостинге (Netlify, Vercel, GitHub Pages).

### 📱 Мобильные приложения (Capacitor)

**Android:**
```bash
npm run build:android
# APK будет в android/app/build/outputs/apk/
```

**iOS:**
```bash
npm run build:ios
# Откройте в Xcode для дальнейшей сборки
```

### 💻 Десктопные приложения (Tauri)

**Windows/macOS/Linux:**
```bash
npm run build:desktop
# Файлы будут в src-tauri/target/release/bundle/
```

**Все платформы сразу:**
```bash
npm run build:all
```

📖 **Подробная инструкция**: см. [BUILD_GUIDE.md](BUILD_GUIDE.md)
