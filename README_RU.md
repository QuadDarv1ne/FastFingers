## 📖 О проекте

**FastFingers** — это веб-платформа для обучения слепой печати, которая сочетает интерактивный интерфейс, игровую механику и возможности искусственного интеллекта для создания персонализированной траектории обучения.

## ✨ Возможности

### 🎯 Интерактивный тренажёр

- Виртуальная клавиатура с подсветкой клавиш
- Мгновенная обратная связь (зелёный — правильно, красный — ошибка)
- Звуковое сопровождение
- Поддержка раскладок: QWERTY, ЙЦУКЕН, Dvorak

### ⚡ Режим спринта

- 60 секунд на максимум
- WPM в реальном времени
- Соревновательный элемент

### 📊 Статистика

- Графики скорости (WPM, CPM) и точности
- Тепловая карта ошибок
- История тренировок
- Лучшие результаты

### 🎮 Геймификация

- Система уровней и достижений
- Ежедневные задания
- XP за каждую тренировку

### 📱 PWA

- Установка как приложение
- Офлайн-режим
- Адаптивность для всех устройств

## 🚀 Быстрый старт

### Требования

- Node.js 18+
- npm 9+

### Установка

```bash
# Клонирование
git clone https://github.com/yourusername/fastfingers.git
cd fastfingers

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
# Создание production-сборки
npm run build

# Предпросмотр
npm run preview
```

## 🧪 Тестирование

```bash
# Unit тесты
npm run test           # Watch режим
npm run test:run       # Однократный запуск
npm run test:coverage  # Отчёт о покрытии

# E2E тесты
npm run e2e            # Все тесты
npm run e2e:headed     # С открытым браузером
```
