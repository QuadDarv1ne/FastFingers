## 🎯 Быстрый старт

### Интеграция с Tilda

**5 минут:**
1. Откройте [`public/tilda-widget.html`](../public/tilda-widget.html)
2. Скопируйте код
3. Вставьте в блок T123 на Tilda
4. Опубликуйте

**Полная версия (iframe):**
1. Соберите: `npm run build`
2. Задеплойте на Netlify/Amvera
3. Вставьте iframe в Tilda

```html
<iframe src="https://yoursite.netlify.app" width="100%" height="750" frameborder="0"></iframe>
```
