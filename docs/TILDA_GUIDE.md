## 📋 Обзор

### Что такое FastFingers?

**FastFingers** — современный веб-тренажёр слепой десятипальцевой печати с:
- ✅ Поддержкой раскладок ЙЦУКЕН, QWERTY, Dvorak
- ✅ Виртуальной клавиатурой с подсветкой
- ✅ Статистикой WPM/CPM/точности
- ✅ Тепловой картой ошибок
- ✅ Системой достижений и прогресса
- ✅ Геймификацией (спринт, тесты скорости)

### Зачем интегрировать с Tilda?

Tilda — популярный конструктор сайтов, но не имеет встроенного тренажёра печати.  
FastFingers можно встроить как:
- **Образовательный модуль** на сайте курсов
- **Интерактивный элемент** для портфолио
- **Сервис для пользователей** на корпоративном портале

## 🚀 Способ 1: Интеграция через iframe

### Принцип работы

```
┌─────────────────────────────────────┐
│         Страница на Tilda           │
│  ┌───────────────────────────────┐  │
│  │          <iframe>             │  │
│  │  ┌─────────────────────────┐  │  │
│  │  │   FastFingers (Netlify) │  │  │
│  │  │   yoursite.netlify.app  │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### Шаг 1: Деплой приложения

См. раздел [Деплой на хостинг](#деплой-на-хостинг)

### Шаг 2: Получение URL

После деплоя вы получите URL вида:
```
https://fastfingers-tilda.netlify.app
```

### Шаг 3: Вставка в Tilda

#### Вариант A: Блок T123

1. Откройте редактор Tilda
2. Нажмите **+** (добавить блок)
3. Выберите **Другое** → **T123 (HTML-код)**
4. Вставьте код:

```html
<iframe 
  src="https://fastfingers-tilda.netlify.app" 
  width="100%" 
  height="750" 
  frameborder="0"
  style="border-radius: 16px; overflow: hidden;"
  title="FastFingers — Тренажёр слепой печати"
  loading="lazy"
></iframe>
```

5. Сохраните и опубликуйте страницу

#### Вариант B: Zero Block

1. Создайте **Zero Block**
2. Нажмите **+** → **Add HTML**
3. Вставьте тот же код iframe
4. Настройте размеры HTML-контейнера
5. Опубликуйте

### Шаг 4: Настройка размеров

#### Адаптивный контейнер

```html
<div style="
  position: relative; 
  width: 100%; 
  padding-bottom: 65%; 
  overflow: hidden; 
  border-radius: 16px;
  background: #0f0f1a;
">
  <iframe 
    src="https://yoursite.netlify.app" 
    style="
      position: absolute; 
      top: 0; 
      left: 0; 
      width: 100%; 
      height: 100%; 
      border: none;
    "
    title="FastFingers"
  ></iframe>
</div>
```

## ⚡ Способ 2: Прямая вставка виджета

### Принцип работы

```
┌─────────────────────────────────────┐
│         Страница на Tilda           │
│  ┌───────────────────────────────┐  │
│  │     Блок T123 (HTML-код)      │  │
│  │  ┌─────────────────────────┐  │  │
│  │  │   FastFingers Widget    │  │  │
│  │  │   (встроенный код)      │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### Что входит в виджет?

✅ **Включено:**
- Тренировка печати (основной режим)
- Виртуальная клавиатура ЙЦУКЕН
- Статистика в реальном времени (WPM, CPM, точность, ошибки)
- Выбор категории и сложности
- Прогресс-бар
- Экран завершения упражнения

❌ **Не включено:**
- Тепловая карта ошибок
- Профиль пользователя
- Достижения и награды
- Спринт и тесты скорости
- Пользовательские упражнения
- Звуковые эффекты
- Сохранение прогресса

### Шаг 1: Подготовка кода

Файл виджета: `public/tilda-widget.html`

### Шаг 2: Копирование

1. Откройте `public/tilda-widget.html` в текстовом редакторе
2. Выделите всё содержимое (Ctrl+A)
3. Скопируйте (Ctrl+C)

### Шаг 3: Вставка в Tilda

1. Откройте редактор Tilda
2. Добавьте блок **T123 (HTML-код)**
3. Вставьте код (Ctrl+V)
4. Сохраните

### Шаг 4: Публикация

**Важно!** Виджет работает только на **опубликованной** странице.  
В предпросмотре скрипты могут не выполняться.

## 🌐 Деплой на хостинг

Для работы iframe вам нужно разместить FastFingers на хостинге.

| Платформа | Сложность | Цена | Серверы в РФ | Авто-деплой |
|-----------|-----------|------|--------------|-------------|
| **Netlify** | ⭐ | Бесплатно | ❌ | ✅ |
| **Amvera** | ⭐⭐ | от 150 ₽/мес | ✅ | ✅ |
| **reg.ru** | ⭐⭐⭐ | от 169 ₽/мес | ✅ | ❌ |

### Quick choice:

- **Нужно быстро и бесплатно** → [Netlify](#netlify)
- **Нужны серверы в РФ (152-ФЗ)** → [Amvera](#amvera)
- **Уже есть домен на reg.ru** → [reg.ru](#regru)

---

### Netlify

#### Способ 1: Netlify Drop (рекомендуется)

**Время:** 3 минуты

1. **Соберите проект:** `npm run build`
2. **Откройте** [app.netlify.com/drop](https://app.netlify.com/drop)
3. **Перетащите папку `dist`** в область загрузки
4. **Скопируйте URL** сайта

#### Способ 2: GitHub + Netlify (автоматический)

1. Запушьте проект на GitHub
2. Войдите на [Netlify](https://app.netlify.com)
3. **Add new site** → **Import an existing project**
4. Подключите GitHub и выберите репозиторий
5. **Build command:** `npm run build`
6. **Publish directory:** `dist`

## 🎨 Настройка и кастомизация

### Изменение цветовой схемы виджета

Откройте `public/tilda-widget.html`, найдите секцию `:root`:

```css
:root {
  --bg-primary: #0f0f1a;      /* Основной фон */
  --bg-secondary: #1a1a2e;    /* Фон карточек */
  --primary: #7c3aed;         /* Акцентный цвет */
  --success: #22c55e;         /* Правильно */
  --error: #ef4444;           /* Ошибка */
}
```

### Изменение раскладки клавиатуры

Найдите `KEYBOARD_LAYOUT` в виджете:

```javascript
// QWERTY
const KEYBOARD_LAYOUT = [
  ['q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'],
  ['a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'],
  ['z', 'x', 'c', 'v', 'b', 'n', 'm']
];
```

## 🐛 Troubleshooting

### Виджет не работает на Tilda

**Решение:**
1. Убедитесь, что страница **опубликована**
2. Проверьте консоль браузера (F12)
3. Попробуйте в другом браузере

---

### iframe не отображается

**Решение:**
1. Проверьте URL в `src` (должен начинаться с `https://`)
2. Убедитесь, что сайт на хостинге доступен
3. Проверьте высоту iframe (минимум 600px)

---

### Стили Tilda ломают виджет

**Решение:**
```html
<style>
  .fastfingers-widget * {
    all: revert;
    box-sizing: border-box;
  }
</style>

<div class="fastfingers-widget">
  <!-- Код виджета -->
</div>
```

## ❓ FAQ

### Вопрос: Можно ли использовать свой домен?

**Ответ:** Да! В настройках хостинга добавьте Custom Domain.

---

### Вопрос: Как обновить виджет после изменений?

**Ответ:**
- **iframe:** `npm run build` → `netlify deploy`
- **Виджет:** Скопируйте обновлённый `tilda-widget.html` в Tilda

---

### Вопрос: Работает ли статистика между сессиями?

**Ответ:**
- **Виджет:** Нет (нет бэкенда)
- **iframe с полной версией:** Да, если подключена авторизация

---

### Вопрос: Поддерживается ли мобильная версия?

**Ответ:** Да, виджет адаптивен.
