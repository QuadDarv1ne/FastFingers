# Интеграция FastFingers с Tilda

## 📋 Обзор

Есть **два способа** интегрировать тренажёр слепой печати FastFingers на страницу Tilda:

| Способ | Сложность | Преимущества | Недостатки |
|--------|-----------|--------------|------------|
| **1. iframe** | ⭐ Легко | Полная функциональность | Требуется хостинг |
| **2. HTML-виджет** | ⭐⭐ Средне | Работает внутри Tilda | Упрощённая версия |

---

## 🎯 Способ 1: Интеграция через iframe (Рекомендуется)

### Шаг 1: Деплой на Netlify

```bash
# Сборка
npm run build

# Деплой (через Netlify Drop)
# 1. Откройте app.netlify.com/drop
# 2. Перетащите папку dist
# 3. Скопируйте URL
```

### Шаг 2: Вставка iframe в Tilda

1. **Откройте редактор Tilda**
2. **Добавьте блок T123 (HTML-код)**
3. **Вставьте код:**

```html
<iframe 
  src="https://your-site.netlify.app" 
  width="100%" 
  height="700" 
  frameborder="0"
  style="border-radius: 12px; overflow: hidden;"
  title="FastFingers Typing Trainer"
></iframe>
```

4. **Опубликуйте страницу**

---

## ⚡ Способ 2: Прямая вставка виджета (HTML-блок)

### Шаг 1: Подготовка

Виджет находится в файле `public/tilda-widget.html`

### Шаг 2: Размещение на Tilda

1. **Откройте `public/tilda-widget.html`**
2. **Скопируйте всё содержимое**
3. **Добавьте блок T123 (HTML-код) на Tilda**
4. **Вставьте скопированный код**
5. **Опубликуйте страницу**
